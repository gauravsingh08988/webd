Text Formatting in HTML

Bold text using the <b> tag.

Strong emphasis using the <strong> tag.

Italic text using the <i> tag.

Emphasized text using the <em> tag.

Underlined text using the <u> tag.

Inserted text using the <ins> tag.

Deleted text using the <del> tag.

Strikethrough text using the <s> tag.

Highlighted text using the <mark> tag.

Small text using the <small> tag.

Superscript: H2O using the <sup> tag.

Subscript: CO2 using the <sub> tag.

Code snippet using the <code> tag.

Preformatted text
using the <pre> tag.

Keyboard input using the <kbd> tag.

Short inline quotation using the <q> tag.

Blockquote for longer quotes
using the <blockquote> tag.

Abbreviation: WHO using the <abbr> tag.

Address:

123 Street, City
using the <address> tag.

Book Title using the <cite> tag.
